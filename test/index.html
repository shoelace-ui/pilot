screen-size-above(size)

Generate a query for screens above a given size.

+screen-size-above('tablet') .above-keyword color green +screen-size-above(500px) .above-size color green
.above-keyword .above-keyword .above-size .above-size

screen-size-below(size)

Generate a query for screens below a given size.

+screen-size-below('wide') .below-keyword color green +screen-size-below(1000px) .below-size color green
.below-keyword .below-keyword .below-size .below-size

screen-size(size)

Generate a query for screens that match a given size.

+screen-size('palm') .palm color green +screen-size('tablet') .tablet color green +screen-size('laptop') .laptop color green +screen-size('desktop') .desktop color green +screen-size('wide') .wide color green
.palm .palm .tablet .tablet .laptop .laptop .desktop .desktop .wide .wide

screen-orientation(orientation)

Generate a query for screens that match a given orientation.

+screen-orientation('portrait') .orientation-portrait color green +screen-orientation('landscape') .orientation-landscape color green
.orientation-portrait .orientation-portrait .orientation-landscape .orientation-landscape
